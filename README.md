# SwiftPackageManagerSample

## SwiftPackageManagerとは

Swiftのソースコードで構成されたパッケージを、依存関係を解決しつつ利用できるようにしてくれる、Apple製のパッケージ管理ツール。
コマンドなど使う必要がないので、導入が簡単。
